# Field Sorting System - User Guide

## How to Access Field Sorting

**Location:** Master Schedule Control Center → **Field Order** Tab

1. Go to **Master Schedule** for your event (Admin → Events → Select Event → Master Schedule)
2. Look for the navigation tabs at the top of the page
3. Click the **"Field Order"** tab (purple button with Settings icon)

## What You Can Do

### 🔄 **Drag and Drop Reordering**
- Simply drag fields up or down to change their order
- The field list shows you the current position numbers (#1, #2, etc.)
- Changes are highlighted in orange until you save them

### 💾 **Save Changes**
- Click **"Save Order"** to apply the new field arrangement
- This order will persist in all scheduling interfaces, especially the Calendar Grid

### 🔄 **Reset to Alphabetical**
- Click **"Reset"** to return fields to alphabetical order
- Useful if you want to start fresh with field ordering

### ℹ️ **Field Information Display**
- Each field shows: Name, Size (11v11, 9v9, etc.), Lights availability, Open/Closed status
- Current sort order number is displayed on the right

## Why Field Order Matters

**Priority Field Strategy:** The scheduling engine consolidates games onto priority fields (typically Fields 12, 13) first, then expands to additional fields. By arranging fields in your preferred order, you ensure:

- **Priority fields appear first** in the Calendar Grid
- **Easier field management** during scheduling
- **Better field utilization** visualization
- **Consistent field naming** across all tournament interfaces

## Field Deletion Issues

If you're getting errors when trying to delete fields:

1. **Check for scheduled games:** Fields with existing games cannot be deleted
2. **Remove games first:** Use the Calendar Interface to move or delete games from the field
3. **Then delete the field:** Once empty, field deletion will work correctly

The system now provides specific error messages explaining why a field cannot be deleted, including how many games are scheduled on it.

## Integration with Master Scheduler

The field order you set here directly impacts:
- **Calendar Grid display order**
- **Field selection dropdowns**
- **Schedule optimization priority**
- **Gap-filling functionality**

Your field arrangement is saved permanently and will be used consistently across all scheduling tools.
