MatchPro Platform: TSX Dependency Issue Documentation

🚨 Critical Issue Summary

The MatchPro application cannot start due to a missing TSX dependency. The tsx package is defined in package.json but is not installed in node_modules, causing the Node.js runtime to fail when trying to import it.

1. Error Details

Primary Error Message

Error: Cannot find package '/home/runner/workspace/node_modules/tsx/index.js' imported from /home/runner/workspace/ at legacyMainResolve (node:internal/modules/esm/resolve:215:26) at packageResolve (node:internal/modules/esm/resolve:841:14) at moduleResolve (node:internal/modules/esm/resolve:927:18) at defaultResolve (node:internal/modules/esm/resolve:1169:11) at ModuleLoader.defaultResolve (node:internal/modules/esm/loader:542:12) at ModuleLoader.resolve (node:internal/modules/esm/loader:510:25) at ModuleLoader.getModuleJob (node:internal/modules/esm/loader:239:38) at ModuleLoader.import (node:internal/modules/esm/loader:472:34) at asyncRunEntryPointWithESMLoader (node:internal/modules/run_main:117:36) at runEntryPointWithESMLoader (node:internal/modules/run_main:146:19) Error Code: ERR_MODULE_NOT_FOUND

Root Cause Analysis

  • TSX package declared but not installed: Listed in devDependencies as "tsx": "^4.19.4" but missing from node_modules
  • Runtime dependency mismatch: Application startup script requires tsx via NODE_OPTIONS but package is unavailable
  • Package manager state inconsistency: package.json and node_modules are out of sync

7. Immediate Resolution Steps

Recommended Fix Sequence

  1. Clear npm cache: npm cache clean --force
  2. Delete node_modules: rm -rf node_modules
  3. Delete package-lock.json: rm package-lock.json
  4. Reinstall dependencies: npm install
  5. Verify tsx installation: npm list tsx
  6. Test application start: npm run dev

Alternative Solutions

  • Manual tsx installation: npm install --save-dev tsx@^4.19.4
  • Switch to ts-node: Replace tsx with ts-node in package.json scripts
  • Use node --loader: Modify NODE_OPTIONS to use TypeScript loader
  • Replit package manager: Use Replit's built-in package installation tools
